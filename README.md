# Computer Graphics 電腦圖學

## Description

使用 opencv-python 學習電腦圖學，包含以下主題：

- Projection: Parallel vs. Prospective
- Translations & Rotations
- Modeling
  - Use triangle and quadrangles 
  - Bezier Curves
    - Recursive Subdivision
- Lighting
  - Shading
    - Flat Shading
    - Gouraud Shading
    - Phong Shading
